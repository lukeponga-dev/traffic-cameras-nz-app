# **App Name**: Kiwi Traffic Watch

## Core Features:

- Camera List: Browse a comprehensive list of traffic cameras throughout New Zealand.
- Live Camera Feeds: View real-time video streams from selected traffic cameras.
- Interactive Map: Display camera locations on an interactive map of New Zealand.
- Favorites: Save frequently viewed cameras to a personal favorites list for quick access.

## Style Guidelines:

- Primary color: Clean blue (#4682B4) to reflect trust and clarity.
- Background color: Off-white (#F0F8FF) for a modern, uncluttered look.
- Accent color: Soft green (#90EE90) to highlight important traffic information.
- Body and headline font: 'Inter' for a modern, neutral, and easily readable experience.
- Use simple, intuitive icons to represent camera locations and other features.
- Design a clean and responsive layout that adapts to different screen sizes.
- Incorporate subtle animations to enhance user interaction, such as loading camera feeds.
